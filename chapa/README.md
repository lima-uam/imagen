# Chapa LIMA

![chapa_lima_sin_texto.svg](chapa_lima_sin_texto.svg)

Diseño de las chapas de la asociación LIMA.
Hay dos archivos:

- `chapa_lima.svg`: diseño de la chapa.
- `chapa_lima_sin_texto.svg`: una copia del anterior pero convirtiendo el texto a un _path_ para evitar problemas de compatibilidad.

Para modificar el diseño se recomienda:

1. Borrar `chapa_lima_sin_texto.svg`.
2. Trabajar sobre `chapa_lima.svg` en Inkscape.
3. Cuando se haya terminado se hace una copia:

```
   cp chapa_lima.svg chapa_lima_sin_texto.svg
```

4. Editar el nuevo archivo en Inkscape:
   1. Seleccionar el texto.
   2. Path -> Object to path (Shift + Ctrl + C).
   3. Guardar.
